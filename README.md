3. Node.js With Passport Authentication by Traversy Media  
[ 使用node.js, express, mongo atlas(mongoose), bootstrap, ejs, passport, bcryptjs ]

在帳號建立時，會檢查email是否已註冊過、email格式是否正確、二次輸入密碼是否正確、密碼長度、欄位有無空白。
